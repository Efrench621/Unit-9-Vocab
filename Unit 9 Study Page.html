Web Form Allows users to input information on a website that can be saved
Controls/Widgest The objects users use to interact with the form
Data Field Stores the data the user inputs
Input Boxes For inserting text and numeric values
Radio Buttons For selecting data values from a set of options
Selection lists For selecting data values from a large list of options
Checkboxes For selecting data values that are yes/no questions
Text Area Boxes For entering larger amounts of text
Spin Boxes For entering integer values within a certain range
Slider Controls For entering numeric values with in a certain range
Calender Controls For selecting date and time values
Color Pickers For choosing color values
Get Method Tells the broswer to take the form data to the end of the URL in the action attribute
Port Method Sends the form data to its own separate data stream
Field Set Group fields that are similar
Legend Defines the content of a field set
Virtual Keyboards Software representations of physical devices
Spinner Control Displays an up and down arrow to increase or decrease the field value by a set amount
Range Slider Limits a numeric field to a range of possible values
Data List A list of data values that a field form can have
Form Buttons A type of form control that performs an action
Command Button Runs a program that affects the content of a page or the actions of the browser
Submit Button Submits a form to the server when clicked
Reset Button Resets the form back to its default state
Validation Process of unsuring that a user has supplied valid data
Server-side Validation Validation occurs on the web server
Client-side Validation Validation occurs in the user's browser
Regular Expression/Regex Description of a character pattern
Inline Validation Immediate data validation and reporting of errors
Focus The state in which an element has been clicked by the user
